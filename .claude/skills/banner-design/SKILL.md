# Banner Design

Banners should communicate one message.

Never overload banners.

---

## Types

AI Summary

Maintenance

Warning

Success

Release Notes

Announcement

Empty State

---

## Layout

Icon

↓

Title

↓

Description

↓

Optional Action Button

---

## AI Banner

Always include

Prediction

Impact

Suggested Action

Example

4 cargoes are predicted to exceed the threshold within 2 hours.

Recommended action

Contact the driver.

Reduce refrigeration temperature.

---

## Style

Rounded

Soft border

Dark background

Minimal colors

No gradients unless necessary.

No large illustrations.

Banners support dashboards.

They never dominate the interface.
